# Zinerva PowerPoint Template Specifications

## Color Palette

### Primary Colors
- **Dark Teal**: `#134e4a` (teal-900)
- **Medium Teal**: `#0f766e` (teal-800)
- **Light Teal**: `#0d9488` (teal-600)
- **Accent Teal**: `#14b8a6` (teal-500)

### Secondary Colors
- **Orange**: `#f97316` (orange-500)
- **Light Orange**: `#fb923c` (orange-400)
- **Pale Orange**: `#fdba74` (orange-300)

### Neutral Colors
- **White**: `#ffffff`
- **Light Gray**: `#f8fafc` (gray-50)
- **Medium Gray**: `#f1f5f9` (gray-100)

## Slide Master Specifications

### Title Slide
- **Background**: Gradient from teal-900 to teal-800
- **Title Font**: White, bold, 44pt
- **Subtitle Font**: teal-200, 28pt
- **Accent Element**: Orange-500 line or shape

### Content Slides
- **Background**: White or gray-50
- **Title Font**: teal-900, bold, 36pt
- **Body Font**: teal-900, 24pt
- **Accent Color**: orange-500 for bullet points and highlights

### Section Dividers
- **Background**: Gradient from teal-900 to teal-800
- **Title Font**: White, bold, 40pt
- **Subtitle Font**: teal-200, 24pt

## Design Elements

### Headers and Footers
- **Header**: Company logo on the left
- **Footer**: Page number in teal-600, right-aligned
- **Divider Line**: 1px teal-200

### Bullet Points
- **Primary Level**: Orange-500 circle
- **Secondary Level**: Teal-600 square
- **Tertiary Level**: Teal-400 dash

### Charts and Graphs
- **Primary Data**: teal-600
- **Secondary Data**: orange-500
- **Tertiary Data**: teal-400
- **Background**: white or gray-50
- **Grid Lines**: teal-200

### Tables
- **Header Row**: teal-900 with white text
- **Alternate Rows**: gray-50
- **Grid Lines**: teal-200
- **Text**: teal-900

## Typography

### Font Family
- **Headings**: Inter (or similar modern sans-serif)
- **Body Text**: Inter (or similar modern sans-serif)

### Font Sizes
- **Title**: 44pt
- **Section Headers**: 36pt
- **Subheaders**: 28pt
- **Body Text**: 24pt
- **Captions**: 18pt

## Layout Guidelines

### Margins
- **Top**: 0.5 inches
- **Bottom**: 0.5 inches
- **Left**: 0.5 inches
- **Right**: 0.5 inches

### Grid System
- Use a 12-column grid for content placement
- Maintain consistent spacing between elements
- Align text and objects to the grid

## Slide Types

### 1. Title Slide
- Full-width gradient background
- Centered title and subtitle
- Optional: Company logo at bottom

### 2. Content Slide
- White background
- Left-aligned title
- Content area with consistent margins
- Optional: Sidebar with teal-900 background

### 3. Two-Column Slide
- White background
- Two equal columns
- Consistent spacing between columns
- Optional: Vertical divider in teal-200

### 4. Image Slide
- White background
- Large image area
- Caption in teal-900
- Optional: Image overlay with teal-900 gradient

### 5. Quote Slide
- Gradient background
- Large quote text in white
- Attribution in teal-200
- Optional: Decorative elements in orange-500

## Animation Guidelines

### Transitions
- Use subtle transitions (fade, push)
- Keep animations professional and minimal
- Avoid distracting effects

### Builds
- Use simple entrance animations
- Maintain consistent timing
- Keep text builds readable

## Best Practices

1. **Consistency**
   - Use the same fonts throughout
   - Maintain consistent spacing
   - Follow the color hierarchy

2. **Readability**
   - Ensure sufficient contrast
   - Use appropriate font sizes
   - Keep text concise

3. **Visual Hierarchy**
   - Use color to emphasize important points
   - Maintain clear heading structure
   - Use white space effectively

4. **Professionalism**
   - Keep designs clean and modern
   - Avoid cluttered layouts
   - Use high-quality images

## Implementation Notes

1. **PowerPoint Setup**
   - Set up the slide master first
   - Create and save color theme
   - Define paragraph styles
   - Set up default layouts

2. **Templates**
   - Create master slides for each type
   - Save as .potx file
   - Include example slides
   - Add usage instructions

3. **Resources**
   - Include logo files
   - Provide icon set
   - Include example images
   - Add font files if needed
